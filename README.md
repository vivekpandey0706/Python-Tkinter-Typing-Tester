# Python-Tkinter-Typing-Tester
A Python desktop application built with Tkinter that measures typing speed and accuracy in real time. Features include dynamic WPM and raw speed calculation, customizable time limits (15–60 seconds), an advanced mode with mutated text strings, and performance visualization using Matplotlib.


## Features:
1) Real time character error highlighting
2) Dynamic WPM and Raw WPM calculation
3) Accuracy tracking based on correct and total characters typed
4) Multiple time modes (15s,30s,45s,60s)
5) Two difficulty modes: Normal mode and Advanced mode (capitalized letters, numbers and punctuation)
6) Random word generation from dataset
7) Performance visualization through WPM vs Time graph and Raw WPM vs Time graph.


## Technologies Used:
1) Python
2) Tkinter (GUI Framework)
3) Matplotlib (performance visualization)
